## Laravel Basic API

composer install


### Transations

#### Get all transactions:
/api/transactions

#### Get one transactions
/api/transaction/[line]

* line would be the .csv line no.


### API User

#### Register API user
api/register

#### Paramaters
* name
* email
* password
* confirm Password

#### Login API user
/login

#### Paramaters
* email
* password
